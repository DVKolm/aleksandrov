# Сайт компании «К.Б. Александров»

Современный адаптивный веб-сайт для компании по производству натуральной молочной продукции.

## Описание проекта

Сайт включает три основные страницы:
- **Главная** - с видео-баннером, информацией о компании и ключевыми преимуществами
- **Ассортимент** - каталог продукции с фильтрацией по категориям
- **Фотогалерея** - Instagram-подобная галерея фотографий производства и продукции

## Как добавить контент

### 1. Добавление видео

Поместите файл видео о производстве в папку `videos/` с именем `production.mp4`.

**Рекомендации:**
- Формат: MP4 (H.264)
- Разрешение: 1920x1080 (Full HD) или выше
- Длительность: 30-60 секунд
- Размер: оптимально до 20MB для быстрой загрузки

### 2. Добавление изображений

#### Для главной страницы:
- `images/poster.jpg` - постер видео (отображается до загрузки видео)
- `images/about.jpg` - фото для раздела "О нас" (производство, команда, и т.д.)

**Рекомендуемый размер:** 1200x800 пикселей

#### Для страницы ассортимента:
Добавьте фотографии продуктов в папку `images/products/`:
- `milk-classic.jpg` - Молоко классическое
- `milk-premium.jpg` - Молоко отборное
- `kefir.jpg` - Кефир
- `yogurt.jpg` - Йогурт натуральный
- `sour-cream.jpg` - Сметана
- `sour-cream-premium.jpg` - Сметана домашняя
- `cottage-cheese.jpg` - Творог классический
- `cottage-cheese-soft.jpg` - Творог мягкий

**Рекомендуемый размер:** 800x800 пикселей (квадрат)

#### Для фотогалереи:
Добавьте фотографии в папку `images/gallery/`:

**Производство (production):**
- `production-1.jpg` до `production-5.jpg` - фото производственного процесса

**Продукция (products):**
- `products-1.jpg` до `products-4.jpg` - фото готовой продукции

**Магазины (retail):**
- `retail-1.jpg` до `retail-3.jpg` - фото продукции на полках магазинов

**Рекомендуемый размер:** 1000x1000 пикселей (квадрат)

### 3. Изменение текстов

#### Контакты
Отредактируйте контактную информацию во всех HTML файлах в секции `<footer>`:
```html
<p>Телефон: <a href="tel:+70000000000">+7 (000) 000-00-00</a></p>
<p>Email: <a href="mailto:info@kb-alexandrov.ru">info@kb-alexandrov.ru</a></p>
```

#### Социальные сети
В футере найдите секцию `.social-links` и замените `#` на ссылки на ваши соцсети:
```html
<a href="https://instagram.com/ваш_аккаунт" aria-label="Instagram">...</a>
<a href="https://vk.com/ваша_группа" aria-label="VK">...</a>
```

#### Раздел "О нас"
В файле `index.html` найдите секцию `.about` и измените текст под вашу компанию.

## Запуск сайта

### Вариант 1: Открытие напрямую
Просто откройте файл `index.html` в браузере двойным кликом.

### Вариант 2: Локальный сервер (рекомендуется)
Для корректной работы видео и некоторых функций рекомендуется использовать локальный сервер.

#### С использованием Python:
```bash
# Python 3
python -m http.server 8000

# Затем откройте в браузере: http://localhost:8000
```

#### С использованием Node.js:
```bash
# Установите http-server глобально
npm install -g http-server

# Запустите в папке проекта
http-server -p 8000

# Откройте в браузере: http://localhost:8000
```

#### С использованием Live Server (VS Code):
1. Установите расширение "Live Server" в VS Code
2. Кликните правой кнопкой на `index.html`
3. Выберите "Open with Live Server"

## Особенности

### Адаптивный дизайн
Сайт полностью адаптирован под все устройства:
- Десктоп (1200px+)
- Планшеты (768px - 1199px)

## Настройка цветовой схемы

Цвета сайта можно легко изменить в файле `css/style.css` в секции `:root`:

```css
:root {
    --primary-color: #2c5f2d;      /* Основной зеленый */
    --primary-light: #3d7d3f;      /* Светло-зеленый */
    --primary-dark: #1e4620;       /* Темно-зеленый */
    --accent-color: #f5c842;       /* Акцентный желтый */
    --text-dark: #2d3436;          /* Темный текст */
    --text-light: #636e72;         /* Светлый текст */
    --bg-light: #f8f9fa;           /* Светлый фон */
    --bg-white: #ffffff;           /* Белый */
}
```

## Требования к изображениям

### Форматы
- JPG для фотографий
- PNG для логотипов и изображений с прозрачностью
- WebP для лучшего сжатия (опционально)

### Оптимизация
Рекомендуется сжимать изображения перед загрузкой:
- **Онлайн:** TinyPNG, Squoosh.app
- **Десктоп:** ImageOptim (Mac), FileOptimizer (Windows)

### Размеры файлов
- Постер видео: до 200KB
- Фото продуктов: до 150KB каждое
- Фото галереи: до 200KB каждое

## Добавление новых продуктов

Чтобы добавить новый продукт в ассортимент:

1. Добавьте изображение в `images/products/`
2. В файле `assortment.html` добавьте новый блок:

```html
<div class="product-card" data-category="категория">
    <div class="product-image">
        <img src="images/products/название.jpg" alt="Название продукта">
        <div class="product-badge">Новинка</div> <!-- опционально -->
    </div>
    <div class="product-info">
        <h3>Название продукта</h3>
        <p class="product-specs">Характеристики</p>
        <p class="product-description">Описание продукта</p>
    </div>
</div>
```

**Доступные категории:**
- `milk` - Молоко
- `kefir` - Кефир и йогурты
- `sour-cream` - Сметана
- `cottage-cheese` - Творог

## Добавление фотографий в галерею

Чтобы добавить новое фото в галерею:

1. Добавьте изображение в `images/gallery/`
2. В файле `gallery.html` добавьте новый блок:

```html
<div class="gallery-item" data-category="категория">
    <img src="images/gallery/название.jpg" alt="Описание">
    <div class="gallery-overlay">
        <div class="gallery-info">
            <span class="gallery-icon">❤️</span>
            <span class="gallery-text">Подпись к фото</span>
        </div>
    </div>
</div>
```

**Доступные категории:**
- `production` - Производство
- `products` - Продукция
- `retail` - В магазинах
