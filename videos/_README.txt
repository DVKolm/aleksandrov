ИНСТРУКЦИЯ: Видео для сайта

Добавьте в эту папку следующее видео:

1. production.mp4
   - Назначение: Видео процесса производства для главной страницы (автовоспроизведение)
   - Формат: MP4 (кодек H.264)
   - Разрешение: 1920x1080 (Full HD) или 1280x720 (HD)
   - Длительность: 30-60 секунд
   - Размер файла: до 20MB (чем меньше, тем лучше для скорости загрузки)
   - Звук: желательно без звука или с тихой фоновой музыкой

СОДЕРЖАНИЕ ВИДЕО:
- Процесс производства молочной продукции
- Современное оборудование
- Контроль качества
- Упаковка продукции
- Работа сотрудников

ТРЕБОВАНИЯ:
- Хорошее освещение
- Стабильная съемка (используйте штатив)
- Качественное изображение
- Динамичная смена кадров (3-5 секунд на сцену)

ОПТИМИЗАЦИЯ ВИДЕО:

Для сжатия видео используйте:

1. HandBrake (бесплатно, Windows/Mac)
   - Пресет: "Web" → "Gmail Medium 5 Minutes"
   - Настройки: H.264, 720p или 1080p

2. CloudConvert (онлайн)
   - https://cloudconvert.com/
   - Конвертация в MP4 с настройками для веб

3. FFmpeg (для продвинутых)
   ffmpeg -i input.mp4 -c:v libx264 -crf 23 -preset slow -c:a aac -b:a 128k production.mp4

СОВЕТЫ:
- Чем короче видео, тем быстрее загрузится сайт
- Можно создать несколько версий: одна для десктопа (Full HD), одна для мобильных (720p)
- Убедитесь, что видео зациклено (начало плавно переходит в конец)
- Избегайте текста и логотипов на видео - они уже есть на странице

АЛЬТЕРНАТИВА:
Если у вас нет видео, можно:
1. Использовать серию фотографий в режиме слайдшоу
2. Заказать видео у видеографа
3. Временно использовать статичное изображение вместо видео
